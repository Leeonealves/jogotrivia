# Projeto Mente Cruzada

Projeto Mente Cruzada was developed by Leone Pereira and Keyla Oliveira. This project was created as our final course project, where we applied the knowledge we acquired during our CTeSP program.

The project consists of a multiple-choice quiz game that includes topics such as cooking, history, geography, curiosities, and more. The main goal of the project is to help people who are curious about Portugal to learn more about its geography, culture, and even a bit of the language.

Users can register, log in, and start playing the game. The game follows a predefined path, starting in the Algarve region and progressing towards the north of Portugal.

The project also includes a backoffice system, where we manage questions and answers, as well as user accounts.
